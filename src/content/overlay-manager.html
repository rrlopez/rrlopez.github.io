How to Use

Get up and running in two simple integration steps without configuring context wrapper nesting hierarchies.

01

Mount the Root Container

Place the OverlayManager host component once at the root of your application — the outermost layout that's always mounted. 

import OverlayManager from '@rrlopez/overlay-manager';
  
  export default function App() {
    return (
      <>
        <MainLayout />
  
        {/* No name prop needed — single global instance */}
        <OverlayManager />
      </>
    );
  }
02

Trigger Modals Globally From Anywhere

Import showModal() anywhere — deeply nested buttons, async API handlers, route loaders — and pass your component along with any props it needs. open and onClose are injected automatically. 

import { showModal } from '@rrlopez/overlay-manager';
  import WarningPrompt from './components/WarningPrompt';
  
  async function handleDelete(id: string) {
    await showModal(WarningPrompt, {
      title: 'Delete this item?',
      description: 'This action cannot be undone.',
      onConfirm: async () => {
        await deleteItem(id);
        return true; // auto-closes the dialog
      },
    });
  }

Code Deep Dive

Built in ~150 lines of clean TypeScript.

No state libraries, no context wrappers. Here are the four mechanisms that make it work:

1. Global Singleton

The constructor assigns OverlayManager.instance = this, making the live React component reachable from any module without context. Hot-reload and StrictMode double-invocation are handled by clearing and re-using the existing instance instead of creating a second one. 

constructor(props) {
    super(props)
    this.state = { overlays: [], isMounted: false }
  
    if (OverlayManager.instance) {
      OverlayManager.instance.clear()
      OverlayManager.instance.updateProps(props)
      return OverlayManager.instance  // re-use
    }
  
    OverlayManager.instance = this
  }

2. Two-Step Close & Animation Guard

Rather than removing the entry immediately, closing sets open = false and lets the modal primitive (Radix, Base UI) play its exit animation before unmounting itself. The entry lingers in state until the next filter sweep removes it. 

delDueToBackButton(id: number) {
    this.setState(state => {
      const overlay = state.overlays.find(o => o.id === id)
      if (overlay) overlay.open = false
      return { ...state }
    })
    return id
  }
  
  delOverlay(id: number) {
    return this.delDueToBackButton(id)
  }

3. SafeDialogGuard — Anti-Tamper

Every rendered overlay is wrapped in SafeDialogGuard, a component that attaches a MutationObserver to document.body. If the DOM node is removed or hidden via inline style, it triggers a forced remount. Global click/keydown listeners provide a second line of defence. 

// On tamper detected:
  forceRemountDialog(id: string) {
    this.setState(state => ({
      dialogs: state.dialogs.map(d =>
        d.id === id
          ? { ...d, id: `${id.split('-')[0]}-${newSegment}` }
          : d
      )
    }))
  }
  
  // Global fallback on any user interaction:
  handleGlobalInteraction = () => {
    for (const dialog of activeDialogs) {
      if (!document.getElementById(`guard-${dialog.id}`))
        this.forceRemountDialog(dialog.id)
    }
  }

4. Bulk-Close: clearModals & delChildModals

clearModals() closes the entire stack at once. delChildModals(id) closes every overlay opened after a given one — useful for multi-step flows where a parent cancel should clean up all child dialogs without knowing they exist. 

// Close everything
  export const clearModals = () =>
    OverlayManager.instance?.clear()
  
  // Close only overlays opened after `id`
  deleteChildOverlay(id: number) {
    const startIndex = this.state.overlays
      .findIndex(o => id === o.id)
  
    const overlays = this.state.overlays.map((o, i) =>
      i > startIndex
        ? { ...o, open: false, key: undefined }
        : o
    )
    this.setState({ overlays })
  }
